Running

1. Make sure [Python 3.10+](https://www.python.org/downloads/) is installed. And change the version of Python in Pipfile with the version you are using in your machine.

2. Install [pipenv]:
   Run the command in the terminal: pip install pipenv

3. Install requirements: pipenv install

4. Split and process the dataset: pipenv run python preprocess.py

5. Run the algorithms: pipenv run python recommend.py
